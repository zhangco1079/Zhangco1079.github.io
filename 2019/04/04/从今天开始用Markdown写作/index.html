从今天开始用Markdown写作

Apr 4, 2019

知道Markdown很久了，一直都不理解为什么Markdown这么麻烦，程序员们和很多文字工作者却对其钟爱有加，昨天发现了Markdown的强大，原来除了文本编辑，Markdown居然可以直接画流程图等各种图标；还可以在文字中直接插入代码；除此之外，直接可以生成Latex，是论文写作利器～真的很强大。决定从今天开始用Markdown来写论文，顺便学习下Latex～～
