安装anaconda的坑

Mar 29, 2019

今天读了python编程之美前四章，发现了还有anaconda这种IDE～

感觉用来教学应该不错，安装简单，界面对文科生比较友好。

  1. anaconda使用参考了：

    https://www.jianshu.com/p/17288627b994?utm_campaign=maleskine&utm_content=note&utm_medium=seo_notes&utm_source=recommendation

  2. 安装完之后，发现命令行无法访问conda，网上搜索之～大半天才搞清楚情况～参考：
    https://www.jianshu.com/p/a6084dc03afa 解决了这个问题。
    但我的电脑是在 ~/.zshrc中插入'export PATH=~/anaconda3/bin:$PATH'
  3. 电脑里有四个python，系统自带python2.7、后来安装的python3.7.2、
    anaconda默认的python3.7.1和anaconda建立的虚拟环境python3.7.1。
    如何配置电脑，可以访问这四个又成了难题。搜索之：
    https://blog.csdn.net/BTUJACK/article/details/81569687
    解决了问题，我的电脑同样在~/.zshrc中插入
    alias python2='/System/Library/Frameworks/Python.framework/Versions/2.7/bin/python2.7'
    alias python3='/Library/Frameworks/Python.framework/Versions/3.7/bin/python3.7'
    解决问题。
  4. 过程中又遇到vim的快捷键问题～继续搜索：
    https://jingyan.baidu.com/article/495ba8410ff14d38b30ede01.html
    学会了I是insert，输入结束后按ESC键之后w保存q退出，最后按回车退出。
    发现vim快捷键也很有用，有空要学一学～～

总结下来 1.命令行的命令，2.vim快捷键，3.conda/pip/homebrew快捷键,

4.python环境配置…等知识都接触了下～虽然浪费了大半天时间，

但感觉这些原来天书一般的东西也没有那么难了～希望每天都能有今天这样大的进步！加油！！！
